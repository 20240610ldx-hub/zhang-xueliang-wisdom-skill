# zhang-xueliang-wisdom-skill

[中文 README](README.md)

A public archive for the Zhang Xueliang (Young Marshal) wisdom skill.

This repository preserves the public version of a historical-person wisdom skill and a research annotation layer that explains its choices and risks. It is not a personality cult project, not a final historical verdict, and not a repository for sensational material.

## Project Spirit

The goal of this skill is to distill Zhang Xueliang's decision temperament, modes of expression, action logic, and lessons from failure while respecting the complexity of historical evidence as much as possible. It aims for transferable methodology, not for reducing a historical person into a slogan, an idol, or a negative label.

This project follows several principles:

- **First-person authenticity comes first**: the "I" in `SKILL.md` should only carry judgments, tone, and self-narration that Zhang Xueliang himself could plausibly have voiced. Later researchers' synthetic judgments, archival inference, and structural criticism should not be forced into his first-person voice.
- **The public version stays restrained**: the public skill preserves enough clues for users to examine themselves and ask deeper questions, but it does not directly hand every reader complex conclusions that require historical-method training to digest.
- **The annotation layer serves calibration**: `PRIVATE-ANNOTATIONS.md` records reliability adjustments, narrative filters, evidentiary doubts, and compressed complexity. Its purpose is not to create the thrill of dark revelations, but to prevent the public version from over-mythologizing or over-simplifying the person.
- **Respect the order in which people learn**: complex historical material requires cross-checking multiple sources, distinguishing primary evidence from secondary narration, and understanding how narrative position shapes the presentation of facts. Without that preparation, premature exposure to controversial conclusions can push readers toward conspiracy thinking or nihilism.
- **Avoid unnecessary harm**: material involving Zhang Xueliang, related figures, descendants, and present-day groups should remain restrained as scholarly discussion. It should not be used for traffic-driven dissemination, personal attacks, or political labeling.

## Files

- `SKILL.md`: the formal installable skill file for Codex/agents. It contains behavior rules, trigger terms, response style, and citation boundaries.
- `PRIVATE-ANNOTATIONS.md`: a publicly archived private annotation layer / research annotation document. It is not an installable skill, not the formal release notes, and not a complete verdict on Zhang Xueliang or related historical events.

## Usage Boundaries

`PRIVATE-ANNOTATIONS.md` exists to reduce misreading, not to manufacture "revelation" material. It records reliability corrections, reasons for editorial choices, evidentiary doubts, compressed complexity, and criteria for selective sharing behind the public skill.

When reading or citing it, follow these boundaries:

- Do not extract fragments from the annotation layer out of context and treat them as the author's final position or as settled historical claims.
- Do not make the skill speak, in Zhang Xueliang's first person, conclusions that only later researchers could have formed.
- For claims about motives, organizational games, archival interpretation, or contradictions in oral history, return to primary sources and serious scholarship for cross-checking.
- The annotation layer is for methodological calibration, research transparency, and risk warnings. It is not for attacking real individuals or organizations, and it is not a political, historical, or moral judgment.
- Do not rewrite the annotation layer into social-media "revelations," short-video scripts, controversy-driven headlines, or decontextualized viral material.
- If you only want to use the skill, read `SKILL.md` first. If you want to study its source choices, read `PRIVATE-ANNOTATIONS.md` afterward.

## Reading Order

Read `SKILL.md` first to understand how the public version turns a historical temperament into a usable judgment framework. Then read the boundaries in this README. Read `PRIVATE-ANNOTATIONS.md` last.

If a reader has not yet built basic historical-method skills, they should not start from the annotation layer. The annotation layer is not a shortcut; it is calibration material for readers willing to carry complexity.
