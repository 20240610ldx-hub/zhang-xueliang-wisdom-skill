# zhang-xueliang-wisdom-skill

张学良（少帅）智慧 skill 的公开归档仓库。

这个仓库保存的是一个人物智慧 skill 的公开版本，以及一份用于解释取舍和风险的研究注释层。它不是人物崇拜项目，不是历史定案，也不是猎奇材料库。

## 项目精神

这个 skill 的目标，是在尽量尊重史料复杂性的前提下，提炼张学良的决策气质、表达方式、行动逻辑和失败教训。它追求的是“可迁移的方法论”，不是把一个历史人物简化成口号、偶像或反面标签。

本项目遵守几个原则：

- **第一人称真实性优先**：`SKILL.md` 中的“我”只能承载张学良本人可能说出的判断、语气和自我叙事。后世研究者的综合判断、档案推理和结构性批评，不应被硬塞进他的第一人称里。
- **公开版保持克制**：公开 skill 只保留足够使用者自检和深入追问的线索，不把需要历史方法论训练才能消化的复杂结论直接抛给所有读者。
- **注释层服务校准**：`PRIVATE-ANNOTATIONS.md` 用来记录信度修正、叙事过滤、史料疑点和被压缩的复杂性。它的目的不是制造“暗面快感”，而是防止公开版过度神化或过度简化。
- **尊重人的成长顺序**：复杂史料需要多源交叉验证、区分一手史料与二手叙述、理解叙事立场如何影响事实呈现。没有这些准备，过早暴露争议性结论容易把人引向阴谋论或虚无主义。
- **避免不必要的伤害**：涉及张学良本人、相关人物、后人和现实群体的内容，应保持学术讨论的克制，不用于流量化传播、人格攻击或政治标签化。

## 文件说明

- `SKILL.md`: 面向 Codex/Agent 使用的正式 skill 文件，包含可安装的行为规则、触发语、回答风格和引用边界。
- `PRIVATE-ANNOTATIONS.md`: 公开存档的“私人注释层 / research annotations”。它不是可安装 skill，不是正式发布说明，也不是对张学良或相关史事的完整定论。

## 使用边界

`PRIVATE-ANNOTATIONS.md` 的存在目的，是减少误读，而不是制造“揭秘”材料。它记录的是公开版 skill 背后的信度修正、取舍理由、史料疑点、被压缩的复杂性和选择性分享标准。

阅读或引用时请遵守这些边界：

- 不要把注释层中的片段脱离上下文，当作作者最终立场或历史断言。
- 不要让 skill 以张学良第一人称说出后世研究者才可能形成的综合结论。
- 涉及人物动机、组织博弈、档案解释和口述史矛盾的内容，应回到原始史料与严肃研究中交叉核验。
- 注释层用于方法校准、研究透明和风险提示，不用于攻击现实个人或组织，也不构成政治、历史或道德裁判。
- 不建议将注释层内容改写成自媒体“揭秘”、短视频脚本、争议性标题或断章取义的传播材料。
- 若只想使用 skill，请优先阅读 `SKILL.md`；若要研究其史料取舍，再阅读 `PRIVATE-ANNOTATIONS.md`。

## 阅读顺序

建议先读 `SKILL.md`，理解公开版如何把人物气质转化为可用的判断框架；再读本 README 的边界说明；最后再读 `PRIVATE-ANNOTATIONS.md`。

如果读者尚未建立基本历史方法论，不建议从注释层开始。注释层不是捷径，而是给愿意承担复杂性的人看的校准材料。

## 这不是

- 这不是张学良完整传记。
- 这不是西安事变最终解释。
- 这不是对任何现实个人、组织或政治立场的攻击材料。
- 这不是鼓励用户挖掘历史暗面来替代全面判断的工具。
- 这不是对 Claude、Anthropic、Codex 或任何模型输出的权威背书。
