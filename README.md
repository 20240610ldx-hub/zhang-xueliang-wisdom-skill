# zhang-xueliang-wisdom-skill

张学良（少帅）智慧 skill 的公开归档仓库。

## 文件说明

- `SKILL.md`: 面向 Codex/Agent 使用的正式 skill 文件，包含可安装的行为规则、触发语、回答风格和引用边界。
- `PRIVATE-ANNOTATIONS.md`: 公开存档的“私人注释层 / research annotations”。它不是可安装 skill，不是正式发布说明，也不是对张学良或相关史事的完整定论。

## 使用边界

`PRIVATE-ANNOTATIONS.md` 的存在目的是减少误读，而不是制造“揭秘”材料。它记录的是公开版 skill 背后的信度修正、取舍理由、史料疑点、被压缩的复杂性和选择性分享标准。

阅读或引用时请遵守这些边界：

- 不要把注释层中的片段脱离上下文，当作作者最终立场或历史断言。
- 涉及人物动机、组织博弈、档案解释和口述史矛盾的内容，应回到原始史料与严肃研究中交叉核验。
- 注释层用于方法校准、研究透明和风险提示，不用于攻击现实个人或组织，也不构成政治、历史或道德裁判。
- 若只想使用 skill，请优先阅读 `SKILL.md`；若要研究其史料取舍，再阅读 `PRIVATE-ANNOTATIONS.md`。
